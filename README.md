# Killer You

Консольный OSINT-инструмент для сбора открытой информации. Поддерживает поиск по Telegram, Discord, доменам, никнеймам, телефонам, IP/MAC и временной почте.

```
╔══════════════════════════════════════════════════════╗
║                 KILLER YOU                           ║
║                OSINT TOOLKIT                         ║
╚══════════════════════════════════════════════════════╝
```

## Возможности

| № | Модуль | Описание |
|---|--------|----------|
| 1 | **Telegram** | Анализ профиля по username |
| 2 | **Discord** | Поиск по User ID |
| 3 | **Domain** | Пассивная разведка домена |
| 4 | **Username** | Поиск ника на 500+ платформах |
| 5 | **Phone** | Международная телефонная разведка |
| 6 | **IP / MAC** | Геолокация и данные по IP или MAC |
| 7 | **Temp Mail** | Создание одноразовой почты и чтение писем |

Результаты каждого запроса выводятся в консоль и сохраняются в папку `kill_results/` в формате JSON.

## Требования

- **Windows** (для готового `.exe`) или **Python 3.10+**
- Интернет-соединение

### Зависимости (при запуске из исходников)

```bash
pip install requests colorama
```

## Быстрый старт

### Вариант 1 — готовый исполняемый файл

1. Скачайте `Kill you.exe` из [Releases](../../releases).
2. Запустите двойным щелчком или из командной строки:

```bash
"Kill you.exe"
```

### Вариант 2 — из исходников

1. Клонируйте репозиторий:

```bash
git clone https://github.com/<username>/killer-you.git
cd killer-you
```

2. Установите зависимости:

```bash
pip install -r requirements.txt
```

3. Укажите API-ключ (см. раздел [Конфигурация](#конфигурация)).
4. Запустите:

```bash
python "Kill you.py"
```

```bash
python "Kill you.py"
```

## Использование

После запуска откроется интерактивное меню. Выберите номер модуля и введите запрашиваемые данные (username, ID, домен, номер телефона и т.д.).

Пример сохранённого результата:

```
kill_results/
├── telegram_username.json
├── discord_123456789012345678.json
├── domain_example.com.json
└── username_nickname.json
```

## Сборка `.exe` (PyInstaller)

```bash
pip install pyinstaller
pyinstaller "Kill you.spec"
```

Готовый файл появится в папке `dist/`.
